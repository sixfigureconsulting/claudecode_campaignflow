## Session 1 — Cialdini Influence Framework

### Feature: Step 4 Sequence Generator

**Goal:** Replace generic AI-written outbound copy with behaviorally-grounded copy based on Robert Cialdini's 7 Principles of Influence, so sequences feel human and psychologically resonant — not AI-generated.

**How it works:**
- User selects a **primary influence principle** before generating sequences
- The selected principle becomes the lead driver of the copy
- All 7 principles are woven into every sequence via the base system prompt
- The selected principle is injected as a `PRIMARY DRIVER` instruction per lead

**The 7 categories:**

| Principle | Tagline | Behaviour |
|-----------|---------|-----------|
| Reciprocity | Lead with genuine value | Opens with tailored insight, mini-audit, checklist. CTA = receive value, not book call |
| Commitment | Micro-ask first, ladder up | First CTA is "Worth a 2-min skim?" — no 30-min meeting asks upfront |
| Social Proof | Show peer results | Leads with specific peer result — segment, firm type, 1–2 concrete metrics |
| Liking | Genuine research first | Opens with observable research about the prospect (hire, post, product launch) |
| Authority | Establish niche credibility | 1–2 compact cues: niche focus + result pattern. No long bios |
| Scarcity | Honest urgency only | Concrete + explained limits only ("3 build slots this quarter"). No fake urgency |
| Unity | Same tribe framing | Shared identity language: "founder-led", "bootstrapped B2B", in-group framing throughout |
